# T15 Fragility Notes

- `stationary_zero_arrow_is_fragile_to_initial_distribution`: Stationary initialization gives zero arrow while basis-state initializations yield infinite arrow at the same horizon.
- `macro_arrow_depends_strongly_on_lens_choice`: At fixed witness and horizon, macro arrow is positive for identity/observe_clock_3 but zero for observe_x_binary.
- `fitted_markov_proxy_can_create_or_distort_macro_arrow`: Fitted first-order macro proxy can create spurious macro arrow or materially underestimate honest macro arrow.
- `threshold_floors_can_introduce_spurious_one_way_support`: Applying floor threshold introduces one-way edges in a chain with originally exact reversible support.
- `object_multiplicity_can_be_metric_artifact`: Cluster count changes from 1 to 2 when switching from TV metric to support-signature metric.
